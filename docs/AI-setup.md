# AI Setup Instructions

There are two ways to use AI with Formulize:

## [Embedded AI Assistant](../ai/setup-embedded)

Use AI right inside your Formulize system — no extra software needed. Just enable it in the _Formulize Preferences_ and then click the __Use AI__ link in the menu. To use the embedded AI assistant with a commercial AI provider, users will need _an API key from the AI provider_.

The embedded AI assistant can also be used with local language models, via <a href="https://ollama.com" target="_blank">Ollama</a>. In this case, you do not need any API keys.

[Read the setup steps...](../ai/setup-embedded)

<hr>

## [External AI Assistant via MCP](../ai/setup-mcp)

You can connect Claude Desktop, Copilot, or another AI assistant to Formulize using MCP (Model Context Protocol). Formulize has a built in MCP server. This gives you access to the full AI experience, including Resources and Prompts, inside a dedicated AI assistant application.

To use Formulize via MCP, users will need _an API key from the Formulize instance that they are connecting with_. Also, the AI assistant needs to support _local_ MCP servers.

[Read the setup steps...](../ai/setup-mcp)
