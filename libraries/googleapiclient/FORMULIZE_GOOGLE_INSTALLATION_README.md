# Google Integration in Formulize

You can use Google for single-signon with Formulize, and you can use special form elements that integrate with your Google drive, and leverage the Google address lookup feature.

## Installation

Download the latest Google PHP API Client from here: [https://github.com/googleapis/google-api-php-client/releases](https://github.com/googleapis/google-api-php-client/releases)

Unzip the contents into this directory. You should end up with a __src__ and __vendor__ folder here, and a __README.md__, etc.

If those files and folders do not end up in this exact folder, you have unziped things into the wrong location.

Contact [info@formulize.org](mailto:info@formulize.org) for help.

## Configuration

After you have installed the files, you need to go to the Formulize settings, and enable Google for authentication.

Contact [info@formulize.org](mailto:info@formulize.org) for help.
