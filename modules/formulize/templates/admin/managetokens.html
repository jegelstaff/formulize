Manage Account Creation Tokens

You can create tokens that let new users join specific groups when they create an account. Share a token with the people you want to be able to join those groups. They can type it in on the sign up page, or you can send them a sign up link with the token already included, for example:

<{$smarty.const.XOOPS_URL}>/signup.php?token=YOURTOKEN

Everyone who signs up is always added to the Registered Users group. A token adds membership in the additional group(s) you select below. A token can grant more than one group at a time.
