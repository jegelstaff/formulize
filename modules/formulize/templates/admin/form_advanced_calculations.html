Before Saving

This special procedure runs before an entry is saved. Values are available as PHP variables using the element handle name. The ID of the current entry (or 'new') is available as $entry_id. On multi-page forms, only the elements which appear on the current page are available, however creating PHP variables for other elements will cause them to be saved.

Enable / Disable Entry Editing

This special procedure allows for a custom condition to be set, to decide whether a user can edit an entry or not. Variables will be available in the code. IE: $entry can be used to check values in an entry, $user for identifying the logged in user, and $allow_editing (true or false) to determine if the user can make changes or not.
