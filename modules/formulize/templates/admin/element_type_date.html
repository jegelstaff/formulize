Default Date

Leave blank for no default date. Use {TODAY} for the current date, or {TODAY+7} for one week in the future, {TODAY-14} for two weeks in the past, etc.

Date Validation (optional)

You can use literal dates: 2021-05-09 or dates relative to the current date: {TODAY+7} or references to another element in this form: {element_handle}
If you use a reference to another element, that element must contain a valid literal date.
